# Multidevice Layout - Flutter

Este es un proyecto desarrollado en Flutter enfocado en la creación de interfaces adaptativas y responsivas para múltiples dispositivos (Smartphone, Tablets, Desktop) utilizando un sistema de breakpoints.

## Acerca del curso

[**Leer libro: Componentes y Animaciones en Flutter**](https://www.desarrollolibre.net/libros/componentes-y-animaciones-en-flutter)

## Características del Proyecto
* **Layouts Adaptativos**: Implementación de diferentes estructuras de UI (`LayoutSM`, `LayoutLG`) basadas en el tamaño de la pantalla.
* **Sistema de Breakpoints**: Uso de una lógica centralizada para determinar el tamaño del dispositivo.
* **Gestión de Estado**: Integración con `Provider` para manejar el estado del layout de forma eficiente.
* **Orientación**: Soporte para cambios de orientación dinámica.

## Tecnologías utilizadas
* **Flutter**: Framework de UI.
* **Dart**: Lenguaje de programación.
* **Provider**: Gestión de estado.

## Instalación

1. Clona este repositorio:
   ```bash
   git clone <url-del-repositorio>
   ```
2. Obtén las dependencias:
   ```bash
   flutter pub get
   ```
3. Ejecuta el proyecto:
   ```bash
   flutter run
   ```
